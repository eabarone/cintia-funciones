Funciones matemáticas

¿Te has preguntado alguna vez cómo crecen los seguidores de una app, cómo se disparan los datos en segundos o cómo saber si algo está creciendo demasiado rápido? 🚀 ¡Bienvenido al mundo de las funciones matemáticas! Son las herramientas secretas para entender y anticipar cambios en la vida real: desde el número de visitas a tu página web favorita hasta el espacio que ocupa una app en tu celular.

En esta unidad vas a conocer tres tipos de funciones que te abrirán los ojos:

  • Polinómicas: muestran crecimientos regulares, como sumar puntos cada día.
  • Exponenciales: reflejan aumentos acelerados, como cuando un video se hace viral.
  • Logarítmicas: te ayudan a interpretar procesos inversos, como descifrar contraseñas o analizar escalas de sonido.

¡Prepárate para descubrir cómo las matemáticas se conectan con la programación y la vida real de una forma divertida y útil! 🤩

Objetivos de aprendizaje

  • 🔎 Comprender qué es una función matemática y por qué es fundamental para modelar situaciones en la vida cotidiana y la tecnología.
  • 📈 Explorar y diferenciar los principales tipos de funciones: polinómicas, exponenciales y logarítmicas, identificando sus características clave y ejemplos prácticos.
  • 🤖 Reconocer cómo las funciones matemáticas permiten describir, predecir y optimizar fenómenos reales y tecnológicos, como el crecimiento de usuarios, el procesamiento de datos o la seguridad digital.
  • 💡 Aplicar el análisis de funciones para resolver problemas prácticos y tomar decisiones informadas en contextos de programación, ciencia y vida diaria.
  • 🌐✨ Valorar la importancia de las funciones matemáticas como puente entre las matemáticas, la tecnología y el mundo real.

¿Qué es una función? 🔄

Podemos definir una función como una regla que relaciona dos cantidades: la primera es la entrada (lo que le das a la función) y la segunda es la salida (lo que obtienes a cambio). A cada valor de entrada le corresponde exactamente un valor de salida.

Ejemplo sencillo:
f(x) = x + 1

Aquí, x es el valor que recibe la función y f(x) es el resultado después de aplicar la "regla" (sumar 1).

Si x = 5: f(5) = 5 + 1 = 6

¡Así de simple! Le das un número a la función, y la función te devuelve otro número siguiendo su regla.

Podemos definir una función polinómica como una fórmula matemática que sirve para relacionar dos cosas que cambian juntas. Por ejemplo:

  • Si aumentas los metros sembrados, también aumenta la cosecha.
  • Si crías más gallinas, también puedes vender más huevos.

Tipos de funciones polinómicas

Nota: En este curso solo vamos a abarcar las funciones polinómicas lineales o de grado 1.

Grado 1: función polinómica lineal

Una función polinómica lineal es una función que crece o decrece de manera constante.

Ejemplo práctico #1

Imagina que estás vendiendo huevos en la finca de tus padres. Tu papá te dice que cada huevo debes venderlo a 10 pesos. Entonces, si vendes 5 huevos, ¿cuál sería la ganancia?

Definimos la función como ganancia(x) = 10x, donde 10 es el valor de cada huevo y x es la cantidad de huevos que vendiste. Si reemplazamos el valor de x por 5 tenemos que:

ganancia(5) = 10 × 5 = 50

Entonces obtuviste una ganancia de 50 pesos.

Si analizamos el ejemplo observamos que las ganancias crecen a medida que vendes más cantidad de huevos.

Ejemplo práctico #2

En este ejemplo continuamos con la misma situación. Imagina que ha pasado al día siguiente y aún tienes los 50 pesos de ganancias contigo y supón que hoy no vendes nada.

Ahora la función se expresaría como ganancia(x) = 10x + 50, donde 50 son las ganancias del día anterior (punto de partida) y x es la cantidad de huevos que has vendido hoy. Si x = 0:

ganancia(0) = 10 × 0 + 50 = 50

Esto refleja que aunque hoy no vendiste nada, tienes los 50 pesos con los que iniciaste el día.

En este caso, cuando x es cero, se observa que el cruce se hace en (x = 0, y = 50), indicando que ya se cuenta con la ganancia del día anterior.

Podemos definir una función exponencial como una función que intenta demostrar qué tanto crece o decrece una cantidad. Estos crecimientos pueden ser muy rápidos porque la variable independiente está en el exponente, haciendo que el valor constante se duplique (o reduzca) según esa variable.

Ejemplo práctico #1

Imagina que tienes una bacteria que se duplica cada hora. Si al inicio tienes 1 bacteria, en una hora tendrás 2, en dos horas 4, en tres horas 8, y así sucesivamente. ¡Cada vez se multiplica por 2!

Esto lo puedes escribir como f(x) = 2ˣ, donde x es el número de horas.

  • Después de 0 horas → f(0) = 2⁰ = 1
  • Después de 1 hora → f(1) = 2¹ = 2
  • Después de 2 horas → f(2) = 2² = 4
  • Después de 3 horas → f(3) = 2³ = 8

Así crece rápidamente la cantidad de bacterias.

¿Cómo se grafica?

Cuando dibujamos la función en una gráfica, al principio crece lento, pero luego sube muy rápido, como una montaña que se vuelve casi vertical. Nunca baja, siempre sube si a > 1.

Podemos definir una función logarítmica como una función que nos permite calcular cuántos pasos o cuánto tiempo se necesita para alcanzar cierta cantidad. Las funciones logarítmicas se relacionan con las exponenciales: mientras la exponencial muestra cómo crece algo, la logarítmica nos dice cuánto tiempo se necesita para ese crecimiento. Se escribe como f(x) = logb(x).

Ejemplo práctico #1

Supón que tienes una gallina, y cada gallina pone 2 huevos al día. Si guardas un huevo por día para incubarlo y nace una nueva gallina, el número de gallinas irá creciendo. Después de un tiempo tienes 8 gallinas. ¿Cuántos días pasaron para llegar de 1 a 8?

La función es f(x) = log2(x). Si x = 8:

f(8) = log2(8) = 3

Porque 2 × 2 × 2 = 8, es decir, se duplicó 3 veces. ¡Pasaron 3 días para llegar a tener 8 gallinas!

Nota: Si la cantidad se triplica o cuadruplica, la base del logaritmo será 3 o 4, respectivamente.

¿Cómo se grafica?

Para graficar la función logarítmica, recuerda que solo funciona para valores positivos de x. Imagina una curva que cruza cerca del punto (1,0) y va aumentando lentamente hacia la derecha a medida que x crece.

3. ¿Cuántos días para llegar a 16 gallinas?
